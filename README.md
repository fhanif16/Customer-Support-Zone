

# 1. What is JSX, and why is it used?
JSX means JavaScript XML.
It lets us write HTML inside JavaScript.
We use it in React because it makes code easy to read and write.


# 2. What is the difference between State and Props?
Props are values sent from parent to child. We cannot change props.
State is data inside a component. We can change state.


# 3. What is the useState hook, and how does it work?
useState is used to store data in a component.
It gives:
a value
a function to change that value
When the value changes, the component updates.

# 4. How can you share state between components in React?
By passing state from parent to child using props (prop drilling).


# 5.How is event handling done in React?
By using events like onClick and onChange.
When an event happens, a function runs and the page updates.
